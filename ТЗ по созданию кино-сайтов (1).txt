2. Задачи
- Разработать каталог контента с поиском и фильтрами.
- Интегрировать несколько существующих плееров с базами (трейлеры, изображения).
- Обеспечить пользовательские функции (рейтинги, комментарии, избранное).
- Реализовать админ-панель для загрузки контента.
- Оптимизировать для SEO и производительности.

3. Функциональные требования
Сайты должны иметь идентичную структуру, с локализацией текста (азербайджанский/узбекский). Контент: фильмы, сериалы, мультфильмы, новости.

3.1. Ключевые модули
- **Поиск и фильтры**: Полнотекстовый поиск по названию, актеру, режиссеру. Фильтры с множественным выбором.
- **Мультимедиа**: Встраивание видео (трейлеры), lazy-loading изображений для постеров.
- **Пользовательские функции**:
  - Регистрация/авторизация (JWT или сессии).
  - Рейтинги (1–5 звезд, средний рейтинг).
  - Комментарии (с модерацией).
  - Избранное/Списки (сохранение контента).
- **Интеграции**:
  - API для данных о кино (TMDB или аналогичный, с локализацией).
  - Социальные сети (поделиться, логин через Google/Facebook).
- **Локализация**: Все тексты на целевом языке. Поддержка RTL для азербайджанского (если применимо). Автоматическое определение языка по IP.

3.2. Административные функции
- Парсинг контента: Формы для добавления фильмов (поля: название, описание, постер, трейлер URL, жанры, актеры).
- Управление пользователями: Блокировка, просмотр активности.
- Аналитика: Просмотры страниц, популярный контент (интеграция Google Analytics).

4. Нефункциональные требования
4.1. Производительность
- Время загрузки страницы: < 3 сек (Core Web Vitals).
- Поддержка 1000+ одновременных пользователей.
- Кэширование (Redis), сжатие изображений.

4.2. Безопасность
- HTTPS, защита от SQL-инъекций, XSS.
- CAPTCHA для комментариев/регистрации.
- GDPR-соответствие (cookies consent).

4.3. Доступность
- WCAG 2.1 (AA уровень): alt-тексты для изображений, клавиатурная навигация.
- Адаптивный дизайн: Bootstrap или аналог (мобильный-first).

4.4. SEO
- Meta-теги, sitemap.xml, robots.txt.
- ЧПУ (URL на азербайджанском/узбекском).
- Schema.org для фильмов (для rich snippets).

5. Дизайн и UI/UX
5.1. Общий стиль
- **Вдохновение референсами**:
  - uzmediaa.net: Темный фон (#000), акценты синий/красный, крупные постеры в сетке.
  - seriallar.net: Минималистичный хедер с мега-меню, бесконечная прокрутка в каталоге.
  - uzmovee.net: Слайдеры с видео, карточки с hover-эффектами.
- **Цветовая схема**: Основной: #121212 (темный), акцент: #E50914 (красный, как Netflix), текст: #FFFFFF.
- **Типографика**: Google Fonts (Roboto для узбекского, аналог для азербайджанского). Заголовки: 24–48px, текст: 14–16px.
- **Иконки**: Font Awesome или SVG.
